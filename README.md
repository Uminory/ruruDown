# ruruDown

B站视频缓存导出工具 —— 将 B站离线缓存无损导出为 MP4/M4A/XML。

## 功能

- 📱 扫描 B站缓存目录，展示可导出视频列表
- 🎬 导出 MP4 视频（FFmpeg 无损合并）
- 🎵 导出 M4A 音频
- 📝 导出 XML 弹幕
- 🔍 按时间/大小排序、搜索筛选
- 📋 BV 号一键复制
- 🎨 批量选中导出

## 系统要求

- Android 8.0 (API 26)+
- 已安装并激活 [Shizuku](https://shizuku.rikka.app/)
